# 🚗 DrivEase
> 장롱면허 보유자를 위한 모바일 운전 시뮬레이터

## 📌 프로젝트 소개
국내 장롱면허 보유자 약 800만 명을 위한
스마트폰 자이로스코프 기반 운전 연습 앱입니다.

기존 운전 앱들이 결과만 알려주는 것과 달리,
**왜 안 되는지 원인 분석과 단계별 가이드**를 제공합니다.

## ✨ 주요 기능
- 📱 자이로스코프 핸들 조작 (폰 기울여서 운전)
- 🚨 실전 돌발상황 시나리오 6종
- 🅿️ 주차 단계별 가이드 시스템
- 📊 항목별 점수 및 원인 분석 피드백
- 🎯 초급 → 중급 → 고급 레벨 시스템
